# Notifications App

Sistema centralizado para la gestión y envío de notificaciones por email a usuarios y administradores.

## Características
- **Templates**: Uso de plantillas HTML para correos profesionales.
- **Triggers**: Integración con señales (signals) de otras apps (ej. al recibir un nuevo pedido).
- **Consola/SMTP**: Configuración flexible según el entorno (dev/prod).

## Tipos de Notificaciones
- Verificación de email.
- Aprobación/Rechazo de cuenta.
- Confirmación de pedido.
- Actualizaciones de estado de pedido.
