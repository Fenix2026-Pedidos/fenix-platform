# Core App

Esta aplicación contiene la configuración global de la plataforma, vistas públicas genéricas y herramientas compartidas.

## Responsabilidades
- **Vistas Públicas**: "Sobre nosotros", "Privacidad" y "Aviso Legal".
- **Búsqueda Global**: Implementación del buscador que atraviesa catálogo y pedidos.
- **Context Processors**: Inyección de datos globales en los templates (ej. logo, configuración del sitio).

## Configuración
Aquí se gestionan (vía Admin) los parámetros globales del sistema, como el idioma por defecto o los emails de contacto.
